# API Routes

## Account

### Register
`POST /api/Account/Register`

- Email
- Password

Returns: JSON

- ModelState.Values
- IdentityResult
- model  + Token / Cookie?

### Login
`POST /api/Account/Login`
- Email
- Password

Returns: JSON

- ModelState.Values
- IdentityResult
- model + Token / Cookie?

### Logout
`GET /api/Account/Logout`


## Lists

### Owned or subscribed list By ID
`GET /api/Lists/{id}`

- Token / Cookie? of list owner or participant

Returns: JSON List with all items

### Edit list
`PATCH /api/Lists/{id}`

- Token / Cookie? of list owner
- Partial list model

### New list
`POST /api/Lists/{id}`

- Token / Cookie?
- List model

### Delete List
`DELETE /api/Lists/{id}`

- Token / Cookie? of list owner


## Items

### Toggle checkmark

`PUT /api/Items/{id}`

- Token / Cookie? of list subscriber

### Edit item
For editing and (un)checking items.

`PATCH /api/Items/{id}`

- Token / Cookie? of item owner
- Partial item model

### New item
`POST /api/Items/{id}`

- Token / Cookie? of list participant
- Item model

### Delete Item
`DELETE /api/Items/{id}`

- Token / Cookie? of item owner


## Users

### Get Logged in User

`GET /api/Users/`

Returns: JSON User

### Get User

`GET /api/Users/{id}`

Returns: JSON User with populated public OwningLists field

### All owned lists
`GET /api/Users/Lists`

- Token / Cookie?

Returns: JSON [List] of logged in user

### All subscribed lists
`GET /api/Users/Subscriptions`

- Token / Cookie?

Returns: JSON [List] of logged in user
