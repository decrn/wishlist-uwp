# API Routes

## Account

### Register
`POST /api/Account/Register`

- Email
- Password

Returns: JSON

- ModelState.Values
- IdentityResult
- model  + Token / Cookie?

### Login
`POST /api/Account/Login`
- Email
- Password

Returns: JSON

- ModelState.Values
- IdentityResult
- model + Token / Cookie?

### Logout
`GET /api/Account/Logout`


## Lists

### All lists (deprecated)
`GET /api/Lists`

Returns: JSON [List]

### All lists (coming)
`GET /api/Lists`

- Token / Cookie?

Returns: JSON [List] of logged in user

### List By ID
`GET /api/Lists/{id}`

- Token / Cookie? of list owner or participant

Returns: JSON List with all items

### Edit list (deprecated)
`PUT /api/Lists/{id}`

- List model

### Edit list (coming)
`PATCH /api/Lists/{id}`

- Token / Cookie? of list owner
- Partial list model

### New list
`POST /api/Lists/{id}`

- Token / Cookie?
- List model

### Delete List
`DELETE /api/Lists/{id}`

- Token / Cookie? of list owner
